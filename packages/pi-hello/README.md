# @rennis23/pi-hello

A minimal pi.dev extension example. Registers a `/hello` command that prints a greeting.

## Install

```bash
pi install npm:@rennis23/pi-hello
```

## Usage

In a pi session:

```
/hello
/hello pi
```

## Development

This package is part of the `pi-mono` workspace. Run tests from the monorepo root:

```bash
npm test
```
